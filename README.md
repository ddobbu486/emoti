# 이모티콘 AI · 32컷 스티커 + 미니 이모티콘 생성기

내 캐릭터 이미지를 올리거나, 성격 / 직업 / 관계 / 취미 / 운동 / 말투를 골라 새 캐릭터를 만들면
Google Gemini 이미지 생성 API로 32컷 색채 완성 스티커(카카오 이모티콘 느낌)와, 텍스트랑 같이 쓰는 미니 이모티콘 아이디어를 만들어주는 정적 사이트입니다.

사이트는 상단 탭으로 **"스티커(32컷)"**와 **"미니 이모티콘"** 두 가지를 오갈 수 있어요.

## 사용 방법 — 스티커(32컷) 탭

1. (선택) 상단 "내 캐릭터 이미지" 영역에 본인이 그린 캐릭터 PNG/JPG 업로드
   - 업로드하면 모든 컷이 이 캐릭터를 기준으로 그려져요
   - 업로드 안 하면 위저드 답변으로 새 캐릭터를 만들고, 첫 컷을 자동으로 기준 이미지로 고정해 일관성을 유지해요
2. 상단에 **Gemini API 키** 입력 후 "저장"
   - 키 발급: https://aistudio.google.com/apikey
   - 키는 브라우저 `localStorage`에만 저장되고 서버로 전송되지 않아요
3. 위저드 따라 종족(사람/동물 종류) → 나이대 → 성별 → 성격 → 직업 → 관계 → 취미 → 운동 → 말투 → 말버릇 → 색감 → 비주얼 포인트까지 선택
4. "컨셉 완성" → 결과 화면에서 컨셉 요약(타겟/받아보는 대상 등) 확인
5. **"턴어라운드 미리보기 생성"**으로 먼저 정면·측면·후면 3면도를 한 장에 뽑아보기 (티콘파이의 "삼면도"와 같은 개념)
   - 마음에 안 들면 "다시 그리기"로 재생성
   - 마음에 들면 "좋아요! 32컷 전부 생성" 클릭 → 이 턴어라운드를 기준으로 32컷이 새로 생성돼요
6. 완료되면 "전체 PNG ZIP 다운로드"로 32장 + 컨셉 텍스트 받기
   - 실패한 컷은 카드에 빨간 "실패"로 표시되고, 마우스를 올리면 에러 메시지를 볼 수 있어요

## 사용 방법 — 미니 이모티콘 탭

**스티커(32컷)와는 컨셉이 완전히 다릅니다.** 실제 카카오 미니이모티콘 인기작들을 분석해 4가지 패턴으로 타입을 나눴어요. 정식 규격은 180×180px PNG, 총 42개입니다.

타입 4가지:
- **① 사물/오브젝트 모음형** — 캐릭터 없이 꽃·과일·동물·음식 등 귀여운 사물 42개를 한 화풍으로 (가장 흔한 패턴)
- **② 한글자·단어 텍스트형** — "헐", "짱", "굳" 같은 짧은 단어를 타이포그래피로 디자인
- **③ 캐릭터 얼굴+장식 혼합형** — 캐릭터 얼굴만 작게 여러 표정 + 하트/별 등 범용 장식
- **④ 도형·숫자 채팅꾸미기형** — 숫자 0~9, 화살표, 체크 등을 패턴 뱃지 형태로

1. 상단 탭에서 "🔡 미니 이모티콘" 클릭
2. 4가지 타입 카드 중 하나 선택 (기본값: 사물/오브젝트 모음형)
3. 스티커 탭에서 캐릭터 컨셉을 이미 완성했다면 자동으로 가져와요(③ 캐릭터형에서만 실제로 참조 이미지로 사용). 입력창에 구체적 테마(예: "식집사 사물형", "헬스 텍스트형")를 적으면 더 좁혀서 생성돼요
4. **"예시 5개 먼저 생성"** 클릭 → 스타일 확인
5. 마음에 들면 **"좋아요! 나머지 37개 마저 생성"** 클릭 → 42개 완성
6. 완료되면 "전체 PNG ZIP 다운로드"로 받기

## GitHub Pages로 배포하기

1. 이 폴더(`index.html`, `style.css`, `app.js`)를 본인 GitHub 저장소에 올리기
2. 저장소 Settings → Pages → Branch를 `main`(또는 `master`)으로 설정, 폴더는 `/ (root)`
3. 몇 분 뒤 `https://<아이디>.github.io/<저장소이름>/` 에서 접속 가능

## 참고

- 100% 정적 페이지라 별도 서버가 필요 없고, 브라우저에서 Gemini API로 직접 호출합니다.
- 모델: `gemini-3.1-flash-image` (나노바나나). 이미지+텍스트 입력을 받아 이미지를 생성하고, 레퍼런스 이미지를 함께 보내면 동일 캐릭터 일관성을 유지하는 데 강합니다.
- 비용은 사용한 만큼 Google 쪽으로 청구됩니다 (Anthropic Console이 아닌 Google AI Studio 결제 기준). 32컷 생성 시 보통 몇백 원~천원대 수준입니다. 정확한 단가는 https://ai.google.dev/pricing 참고.
- 본인만 쓸 용도라 키를 브라우저에 저장하는 방식이 가장 간단합니다. 여러 사람이 같이 쓰는 공개 사이트로 만들려면 키를 서버(예: Cloudflare Workers)에 숨기는 구조로 바꿔야 합니다.
- 일부 컷은 모델 정책상 생성 실패할 수 있어요(카드에 "실패"로 표시). 그 컷만 다시 시도하고 싶으면 새로고침 후 재생성하거나, 추후 "개별 재시도" 기능을 추가할 수 있어요.
