# 이모티콘 AI · 32컷 가이드 도안 생성기

성격 / 직업 / 관계 / 취미 / 운동 / 말투를 고르면 캐릭터 컨셉을 만들고,
Claude API로 32컷 라인아트 가이드 도안(따라 그리기용)을 생성해주는 정적 사이트입니다.

## 사용 방법

1. 사이트 접속 후 상단에 **Anthropic API 키**를 입력하고 "저장" 클릭
   - 키 발급: https://console.anthropic.com/settings/keys
   - 키는 브라우저 `localStorage`에만 저장되고, 본인 외에는 아무도 못 봐요 (서버로 전송 안 됨)
2. 위저드 따라 성격 → 직업 → 관계 → 취미 → 운동 → 말투 → 비주얼 포인트 선택
3. "컨셉 완성" 누르면 결과 화면으로 이동
4. "32컷 전부 생성" 클릭 → 순서대로 캐릭터 일관성 유지하며 32장 생성
5. 완료되면 "전체 ZIP 다운로드"로 SVG 32장 + 컨셉 텍스트 받기
6. SVG를 그림판/프로크리에이트/포토샵 등에서 열어 따라 그리기

## GitHub Pages로 배포하기

1. 이 폴더(`index.html`, `style.css`, `app.js`)를 본인 GitHub 저장소에 올리기
2. 저장소 Settings → Pages → Branch를 `main` (또는 `master`)으로 설정, 폴더는 `/ (root)`
3. 몇 분 뒤 `https://<아이디>.github.io/<저장소이름>/` 에서 접속 가능

## 참고

- 이 사이트는 100% 정적 페이지라 별도 서버가 필요 없습니다.
- API 호출은 브라우저에서 Anthropic API로 직접 이루어집니다 (`anthropic-dangerous-direct-browser-access` 헤더 사용).
- 본인만 쓸 용도이므로 키를 브라우저에 저장하는 방식이 가장 간단합니다.
  여러 사람이 같이 쓰는 공개 사이트로 바꾸려면 키를 서버(예: Cloudflare Workers)에 숨기는 구조로 바꿔야 합니다.
- 모델: `claude-sonnet-4-6`. 비용은 32컷 생성에 보통 몇십 원~몇백 원 수준입니다 (입력 $3/출력 $15 per MTok 기준, 컷당 토큰이 적어 저렴합니다).
